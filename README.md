This is an Online Pizza Ordering Platform designed using MERN Stack
How To Use this Project:

1. Add .env file with following configurations

        MONGO_URI=
        
        NODE_ENV=production
        
        CLIENT_URL=
        
        PORT=8000
        
        BRAINTREE_ID=
        
        BRAINTREE_PRIVATE=
        
        BRAINTREE_PUBLIC=
        
        JWT=abc@123
        
2. Run "npm install"
3. Run "npm run dev" command to execute this project
